1. Add a blank gedit index.html file in your repository
  2. Fill the file with some text and for stylesheets add main.css
  3. In main.css add a border,font-style,etc.
  4. After this validate your index.html and main.css file.
  5. To view your inex.html file type in localhost:8000 for a specific file in your repository type /filename
  6. Done with all listed above then open sftp with terminal.
  7. In sftp cd into public_html.
  8. Now put index.html and main.css into sftp.
  9. To view in sftp type cmis-hsteacher/~username/ for specific file same as before.
  10. Voila! your done.
